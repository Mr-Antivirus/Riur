# Riur (Ree-Err)
A boilerplate based on React.js, Immutable.js, Universal Javascript, and Redux.

### Installation
```sh
$ git clone https://github.com/Mr-Antivirus/Riur.git riur
$ cd riur
$ rm -rf .git
$ git init
```

### Roadmap
Here is a list of planned features:
* Universal Auth (JWT)
* Database
* React Router
* Hot Reloading
* Immutable Redux Store
* Unit Tests!
